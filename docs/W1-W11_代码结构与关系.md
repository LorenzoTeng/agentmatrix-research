## 各模块关系

```
用户说"中盘股动量"
        │
        ▼
auto_factor_miner.py ─── 调 OpenAI 生成 Qlib 表达式
        │
        ▼
mining_bridge.batch_verify() ─── 表达式能解析吗？能映射到 GM 吗？
        │                          状态: PARSED / PENDING_JQ / NC
        ▼
feedback_to_prompt() ─── 淘汰原因 → 文本 → 注入下一轮 prompt
        │
        ▼
expression_to_spec() ─── 通过的表达 → FactorResearchSpec 模板
        │
        ▼
specs.py ─── 注册进 jq_gm 因子库 (215 个)
        │
        ▼
factors.py ─── compute_jq_gm_factors()
        │                │
        │         ┌──────┴──────┐
        │         ▼             ▼
        │     stub (NaN)    GM SDK (VM)
        │                          │
        ▼                          ▼
nc_classifier.py ─── 跨引擎对比前过滤不可比因子 (129 个已分类)
        │
        ▼
diagnostics.py ─── 12 种检测器定位 MISMATCH 根因
        │
        ▼
check_regression.py ─── 改代码后自动检测输出是否变化
                        Path A (stub): 25 表达式
                        Path B (GM):   308 因子
```
